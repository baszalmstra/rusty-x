Rusty-x
=======

A simple snippet manager to find your snippets with colorized terminal output

Usage
=====

To find files:

```bash
cargo run <KEYWORDS>
```

To add a snippet with a keyword line:
```bash
cargo run <KEYWORDS> --add
```
